Budget setup flow — how you set limits

1. TAP OVERALL CARD
Tap the Total Budget card → bottom sheet slides up → number input → Save
2. TAP CATEGORY ROW
Tap any category bar → same bottom sheet → set or clear that category's limit
3. ADD NEW CATEGORY
Tap "+ Set limit for another category" → category picker → amount → Save
